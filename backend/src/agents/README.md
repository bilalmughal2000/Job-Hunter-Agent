# agents/

Pluggable, independently testable AI agents. Each agent exposes a narrow
interface and is swappable in isolation (see spec §"AI Agents").

- `search/` — Search Agent (Phase 3)
- `dedup/` — Deduplication Agent (Phase 3)
- `matching/` — AI Matching Agent (Phase 5)
- `resume-optimizer/` — Resume Optimizer Agent (Phase 5)
- `cover-letter/` — Cover Letter Agent (Phase 5)
- `notification/` — Notification Agent (Phase 8)
- `analytics/` — Analytics Agent (Phase 9)
- `resume-application/` — Resume & Application Agent (Phase 4/6)
